# Guidelines

## Code Quality Standards
### Princípio da Memória Muscular (Trustware)
- **Operadores são Músculos**: Agentes de execução e workers não devem conter lógica de decisão.
- **Execução Cega**: A eficiência vem da repetição otimizada, não da reinterpretação.
- **Referência**: Ver `docs/Vault/CONSTITUICAO/PRINCIPIO_MEMORIA_MUSCULAR_OPERACIONAL.md`.

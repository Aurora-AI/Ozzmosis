# Cinematic Commerce - Estados (Rodobens Wealth)

## Estados (alto nivel)
1. HOME
2. PORTAL_SELECT (Casa | Carro | Familia)
3. DISCOVERY (LifeMap / Intencao)
4. CATHEDRAL (densidade premium aumenta, sem barreira)
5. PROPOSAL_VIEW (Essential | Wealth | Legacy)
6. SIGNATURE_TABLE (aceite / prova de entendimento)
7. NEGOTIATION (humano + automacoes)
8. CLOSED_WON / CLOSED_LOST

## Transicoes (regras)
- HOME -> PORTAL_SELECT: sempre
- PORTAL_SELECT -> DISCOVERY: sempre
- DISCOVERY -> PROPOSAL_VIEW: somente com LifeMap persistido (Lei: verdade first-class)
- PROPOSAL_VIEW -> SIGNATURE_TABLE: somente com proposal gerada (engine)
- SIGNATURE_TABLE -> NEGOTIATION: somente apos aceite forense (snapshot + fingerprint)

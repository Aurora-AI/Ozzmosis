# Mock Integration Environment — Ademicon (MRS)

Objetivo:
Gerar dados sintéticos coerentes com o modelo MRS (Maturity, Rhythm, Safety) para alimentar Chronos, Conductor e o Dashboard do Edital.

Modelo MRS:
- Maturity (M): disciplina e uso de ferramentas/rotina consultiva.
- Rhythm (R): cadência de interação e sinais de ritmo comercial.
- Safety (S): estabilidade da jornada e risco de churn percebido.

Aviso:
> “Estes dados são sintéticos e existem apenas para demonstração do edital.”

# Vault - Rodobens Wealth (SSOT)

Este diretorio e a Fonte Unica da Verdade (SSOT) para o dominio Rodobens Wealth.

## Estrutura
- `sources/`        -> PDFs e fontes brutas (somente leitura, sem edicao manual)
- `knowledge/`      -> Markdown derivado (gerado) + curadoria
- `policies/`       -> politicas deterministicas (YAML) para Trustware / validacoes
- `diagrams/`       -> diagramas e estados (Cinematic Commerce)
- `os/`             -> Ordens de Servico executadas (fechamentos canonicos)
- `evidence/`       -> evidencias tecnicas (logs, checksums, manifests)

## Regra de Ouro
- Verdade e evidencia vivem aqui.
- Codigo executavel vive fora do Vault.

# Vault Ademicon (SSOT)

Este diretório é a **Fonte Única da Verdade (SSOT)** do projeto INNOVA ADEMICON (Chamamento 009/2025) dentro do Ozzmosis.

Regra operacional:
> Quod non est in Vault, non est in mundo.  
> (O que não está no Vault, não existe.)

Escopo:
- Registrar a doutrina, os estados da jornada e as políticas decisórias
- Conter schemas e exemplos mínimos para geração de mocks
- Servir como base estática para Conductor / Shield / Mock Integration Environment

Importante:
- Este Vault é **verdade em disco**.  
- Nada aqui é “discussão”: é referência operacional.

Estrutura:
- 00_MANIFESTO: Doutrina e framing do produto
- 01_STATES: Máquina de estados (SSOT)
- 02_MOCKS: Schemas e exemplos de dados sintéticos
- 03_POLICIES: Políticas formais do Conductor
- 04_DASHBOARD: Especificação de leitura e UI mínima

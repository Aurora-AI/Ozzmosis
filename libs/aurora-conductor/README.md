# Aurora Conductor (libs/aurora-conductor)

Aurora Conductor é a camada de **síntese e governança** que traduz **contexto constitucional**
em **artefacts de plano e execução**.

Nesta fase (0.1.x), o foco é:
- Perímetro seguro de filesystem (deny-by-default)
- Carregamento de contexto persistente (Constituição/Manual/Produto)
- Fluxo mínimo: Spec → Plan → Policy Review → Artifact (dry-run)

Sem integrações externas nesta etapa.
